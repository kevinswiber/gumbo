# Task 1.2: Add Face Extent Helpers to NodeBounds

## Objective
Add methods to `NodeBounds` that return the usable range of positions along a given face, for use by the spreading algorithm.

## Location
Modify: `src/render/shape.rs`

## Implementation

Add methods to the `NodeBounds` impl block:

```rust
impl NodeBounds {
    /// Returns the range of positions (start, end) along a face where edges can attach.
    /// For Top/Bottom faces: returns (left_x + 1, right_x - 1) — horizontal range excluding corners.
    /// For Left/Right faces: returns (top_y, bottom_y) — vertical range (full height, corners are less
    /// of an issue for vertical faces since nodes are wider than tall).
    pub fn face_extent(&self, face: NodeFace) -> (usize, usize) {
        match face {
            NodeFace::Top | NodeFace::Bottom => {
                let start = self.x.saturating_add(1);
                let end = (self.x + self.width).saturating_sub(2);
                (start, end.max(start))
            }
            NodeFace::Left | NodeFace::Right => {
                let start = self.y;
                let end = self.y + self.height.saturating_sub(1);
                (start, end.max(start))
            }
        }
    }

    /// Returns the fixed coordinate for a face (the y for Top/Bottom, x for Left/Right).
    pub fn face_fixed_coord(&self, face: NodeFace) -> usize {
        match face {
            NodeFace::Top => self.y,
            NodeFace::Bottom => self.y + self.height.saturating_sub(1),
            NodeFace::Left => self.x,
            NodeFace::Right => self.x + self.width.saturating_sub(1),
        }
    }
}
```

## Context
- `NodeFace` will be defined in `intersect.rs` (task 1.1) — import it
- `NodeBounds` already has `x`, `y`, `width`, `height` fields and helpers like `center_x()`, `center_y()`, `top()`, `bottom()`, `left()`, `right()`
- The 1-cell margin on horizontal faces avoids placing edges at corners where the border characters are (`┌`, `┐`, `└`, `┘`)
- For vertical faces, the full height is usable since the border characters run the full height

## Acceptance Criteria
- [ ] `face_extent()` returns correct ranges for all four faces
- [ ] `face_fixed_coord()` returns the correct row/column for each face
- [ ] Handles small nodes gracefully (start <= end always)
