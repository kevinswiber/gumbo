### Step 3: Manually verify cycle-containing fixtures

Run each fixture and visually inspect that backward edges are present and sensible:

```bash
cargo run -- tests/fixtures/simple_cycle.mmd
cargo run -- tests/fixtures/multiple_cycles.mmd
cargo run -- tests/fixtures/complex.mmd
```

For each fixture, verify:
- All nodes are rendered
- Forward edges have correct arrows
- Backward edges are visible and have correct arrow direction
- No characters overlap incorrectly
